# NoShowCrossDomain
Predictive modelling approached across two domains: Flight bookings and Medical appointments
Healthcare Appointment Dataset used here: https://www.kaggle.com/datasets/joniarroba/noshowappointments, https://www.transtats.bts.gov/Distance.aspx
Airline Flight Booking Dataset used here: https://www.kaggle.com/datasets/keatonballard/synthetic-airline-passenger-and-flight-data
Data Preprocssing steps were applied
Handling inconsistent or incorrect feature values
Feature engineering
Encoding categorical features
Target encoding for high-cardinality features
Train/test 80:20 split with stratified sampling
ML Models: Logistic Regression, Random Forest, Decision Tree, XG Boost
Model Calibration applied - Platt scaling, Isotonic Regression
Evaluation Metrics - classification on Precision, F1-socre, ROC-AUC
Probability evalaution on Mean Squared Error (MSE)
